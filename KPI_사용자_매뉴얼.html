1. 전체 흐름

KPI는 먼저 항목을 만들고, 월별 목표값을 설정한 뒤, 매월 실적과 증빙을 등록하고, 승인 절차를 거쳐 관리합니다. 모든 실적이 정리되고 더 이상 관리할 변경이 없으면 권한자가 최종 완료 처리합니다.

1. KPI 항목 추가분류, 세부내용, B/M, 측정 기준, 목표값, 가중치, 담당자 입력
2. 월별 목표 설정월별 목표값과 목표 말풍선 입력
3. 실적 등록월 선택 후 실적값, 코멘트, 그래프 말풍선, 첨부파일 저장
4. 승인그룹장 1차 승인, 팀장 최종 승인 또는 권한자 즉시 반영
5. 관리/수정필요 시 실적 수정, 지연 월 추가, 말풍선 수정
6. 최종 완료저장 후 최종 완료 처리, 부서 화면 달성률 영역에 완료 표시
핵심 원칙: KPI 그래프는 월별 목표값을 기준으로 생성됩니다. 목표값을 어떻게 입력했는지가 이후 실적 입력, 달성률, 그래프 모양에 그대로 영향을 줍니다.

3. KPI 항목 추가/수정

부서 상세 화면에서 KPI 항목 추가를 누르면 새 KPI를 만들 수 있습니다. 이미 등록된 KPI는 권한자에게 KPI 수정 버튼으로 표시됩니다.

3-1. 기본 정보 입력

입력 항목 입력 방법 예시
KPI 대분류 / 소분류 항목을 찾기 쉽게 분류명을 입력합니다. 품질 / 고객 클레임, 생산성 / 설비 제작
세부 내용 KPI가 무엇을 관리하는지 설명합니다. 신규 설비 제작 일정 준수, 불량률 개선
B/M 전년 실적, 기준값, 벤치마크, 신규 여부를 입력합니다. 80%, 0.2, 전년 82.5, 신규제작
평가 유형 수치로 평가하면 정량, 진행률/완료 여부 중심이면 정성 선택 정량 평가, 정성 평가
측정 기준 그래프와 달성률 계산에 사용할 단위를 선택합니다. %, PPM, 건, 명, k, 억, 시간, UPH
측정 방향 실적이 높을수록 좋은지, 낮을수록 좋은지 선택합니다. 매출: 높을수록 좋음, 불량률: 낮을수록 좋음
계산 기준 월별 실적을 당월 단독으로 볼지, 누적으로 볼지 선택합니다. 매월 불량률: 당월 단독, 9월까지 5건 완료: 누적 계산
가중치 부서 내 KPI 중요도를 1~100 사이 숫자로 입력합니다. 10, 20, 30
담당자 해당 KPI를 관리할 담당자를 입력합니다. 홍길동, 1111

3-2. B/M 입력 시 그래프 표시

  • B/M에 숫자가 있으면 측정 기준에 맞춰 표시됩니다. 예: 측정 기준이 억이면 0.2억으로 표시됩니다.
  • B/M이 신규, 신규제작처럼 숫자가 아닌 텍스트이면 라벨은 텍스트 그대로 보이고, 그래프 막대 높이는 최종 목표값만큼 표시됩니다.
  • B/M 막대는 그래프의 1월 왼쪽에 별도 기준 막대로 표시됩니다.

4. 월별 목표값 설정

KPI 항목 추가/수정 화면 오른쪽의 월별 목표값 설정에서 월별 목표값과 목표 말풍선을 입력합니다. 목표값은 그래프의 빨간 점선 기준이 됩니다.

4-1. 당월 단독

각 월의 목표와 실적을 그 달 하나만 놓고 비교합니다. 매월 독립적으로 관리하는 지표에 적합합니다.

예시: 매월 불량률 목표를 3월 2%, 4월 1.8%, 5월 1.5%처럼 관리하는 경우
  • 3월 목표값: 2
  • 4월 목표값: 1.8
  • 5월 목표값: 1.5

4-2. 누적 계산

각 월에 입력하는 값은 “그 달에 추가로 달성해야 하는 양”입니다. 그래프에서는 1월부터 해당 월까지의 합계가 목표선으로 표시됩니다.

예시: 9월까지 총 5건 완료해야 하는 KPI
  • 1월 목표값: 1건
  • 4월 목표값: 2건
  • 6월 목표값: 1건
  • 9월 목표값: 1건
최종 목표값은 자동으로 5건이 됩니다. 그래프 목표선은 1월 1건, 4월 3건, 6월 4건, 9월 5건처럼 누적으로 올라갑니다.

4-3. 목표 말풍선

목표 말풍선은 월별 목표값 옆에 입력하는 짧은 설명입니다. 목표선 근처에 표시되어 해당 월 목표의 의미를 보여줍니다.

  • 입력 예시: 검토, 제작, 책임, 양산
  • 미입력 시 그래프에 표시되지 않습니다.
  • 입력 후 지우고 저장하면 표시되지 않습니다.
  • 실적 말풍선과 겹치지 않도록 목표점에서 점선으로 연결되어 별도 위치에 표시됩니다.
목표 말풍선은 “목표에 대한 설명”입니다. 실제로 왜 지연됐는지, 어떤 일이 있었는지 같은 실적 내용은 실적 등록 화면의 코멘트 또는 그래프 말풍선에 입력합니다.

5. 그래프와 말풍선 읽는 법

목표선

빨간 점선은 월별 목표값입니다. 누적 계산 KPI는 월별 추가 목표값이 누적되어 표시됩니다.

실적 막대

파란 막대는 승인 반영된 실적입니다. 저장했지만 승인 전인 경우 상태에 따라 그래프 반영이 다르게 보일 수 있습니다.

B/M 막대

B/M 기준값은 1월 왼쪽에 별도 막대로 표시됩니다.

도전 목표

도전 목표를 달성한 월은 금색 계열로 강조됩니다.

5-1. 그래프 축 기준

  • 높을수록 좋은 % KPI는 100%가 최대 기준입니다. 라벨이 잘리지 않도록 그래프 공간은 위쪽에 여유를 두지만, 눈금은 20%, 40%, 60%, 80%, 100% 기준으로 보입니다.
  • 낮을수록 좋은 KPI는 목표치가 그래프 중간 근처에 오도록 표시됩니다.
  • 건, 억, 명, 시간, UPH 등 수치형 KPI는 목표값과 실적값의 단위에 맞춰 표시됩니다.

5-2. 말풍선 종류

종류 입력 위치 그래프 표시 위치 용도
목표 말풍선 KPI 항목 추가/수정 > 월별 목표값 설정 빨간 목표선 근처 그 달 목표의 계획 단계 설명
실적 말풍선 실적 등록/수정 > 그래프 말풍선 실적 막대 근처 그 달 실적의 짧은 사유 또는 메모
코멘트 실적 등록/수정 > 세부내용 / 코멘트 하단 상세 영역 상세 설명, 이슈, 조치 내용 기록
같은 월에 목표 말풍선과 실적 말풍선이 모두 있어도 서로 겹치지 않도록 위치를 벌리고 점선으로 연결됩니다.

6. 실적 등록/수정

KPI 항목을 클릭하거나 실적 등록 버튼을 누르면 상세 모달이 열립니다. 권한이 있으면 상단 또는 오른쪽 패널에서 실적을 등록/수정할 수 있습니다.

6-1. 기본 절차

  1. 부서 상세 화면에서 KPI 항목을 선택합니다.
  2. 실적 등록 또는 실적 수정을 누릅니다.
  3. 월 선택에서 입력할 월을 선택합니다.
  4. 실적값을 입력합니다. KPI 유형에 따라 달성률, 실제 수치, 완료 여부 등이 표시됩니다.
  5. 필요하면 세부내용 / 코멘트를 입력합니다.
  6. 그래프 위에 짧게 보일 내용은 그래프 말풍선에 입력합니다.
  7. 증빙 파일이 필요하면 첨부합니다.
  8. 저장을 누릅니다.

6-2. 저장 후 상태

저장한 사람 저장 후 상태 다음 처리자
프로 · 선임 · 책임 · 수석 1차 승인 대기 그룹장
그룹장 최종 승인 대기 팀장
팀장 승인 완료로 즉시 반영 추가 승인 불필요
관리자 권한에 따라 즉시 반영 또는 승인 처리 가능 관리자 판단

6-3. 수정 가능 여부

  • 제출 전 상태의 월은 일반 실적 등록자가 수정할 수 있습니다.
  • 승인 대기 또는 승인 완료된 월은 그룹장, 팀장, 관리자만 수정할 수 있습니다.
  • 수정 후 저장하면 승인 상태가 다시 변경될 수 있으므로, 승인자에게 알리는 것이 좋습니다.

7. 승인과 반려

실적은 등록자, 그룹장, 팀장 흐름으로 검토됩니다. 승인자는 KPI 상세 모달의 승인 처리 영역 또는 실적 승인 관리 화면에서 처리합니다.

제출 전아직 승인 요청 전이거나 임시 상태
1차 승인 대기일반 사용자가 저장한 실적, 그룹장 검토 필요
최종 승인 대기그룹장 승인 후 팀장 검토 필요
승인 완료그래프와 달성률에 반영되는 최종 상태

7-1. 반려되는 경우

  • 증빙 파일이 없거나 내용이 맞지 않는 경우
  • 실적값이 목표 또는 첨부자료와 맞지 않는 경우
  • 코멘트 설명이 부족한 경우
  • 월을 잘못 선택한 경우
반려되면 반려 사유를 확인하고 실적을 수정한 뒤 다시 저장해야 합니다.

8. 최종 완료

최종 완료는 해당 KPI 항목의 실적 관리가 끝났음을 표시하는 기능입니다. 완료 처리되면 부서 화면의 달성률 영역에 완료로 표시되고, 대시보드의 최종 완료 KPI 수에도 반영됩니다.

8-1. 언제 최종 완료하나요?

  • 해당 KPI의 최종 목표월까지 실적 등록과 증빙 첨부가 끝났을 때
  • 필요한 승인 또는 내부 확인이 끝났을 때
  • 더 이상 지연 월을 추가하거나 실적을 보완할 계획이 없을 때

8-2. 왜 실적 저장 화면에서 하나요?

최종 완료는 마지막 실적, 코멘트, 첨부파일까지 함께 저장된 뒤 처리되어야 합니다. 그래서 상세 모달의 저장 후 최종 완료 버튼을 사용합니다.

  1. 최종 실적을 입력합니다.
  2. 필요한 코멘트와 그래프 말풍선을 입력합니다.
  3. 증빙 파일을 첨부합니다.
  4. 저장 후 최종 완료를 누릅니다.
  5. 확인창에서 최종 완료를 승인합니다.
최종 완료는 “승인 완료”와 다릅니다. 승인 완료는 특정 월 실적 검토가 끝났다는 뜻이고, 최종 완료는 KPI 항목 전체 관리가 끝났다는 뜻입니다.

9. 지연 월 추가

일정 지연으로 기존 종료월 이후에도 실적을 관리해야 할 때 권한자가 지연 월을 추가할 수 있습니다. 자동으로 +3개월을 보여주는 방식이 아니라, 필요한 경우에만 한 달씩 추가합니다.

  • 가능 권한: 관리자, 그룹장, 팀장, 그룹장/팀장 겸임
  • 위치: 실적 등록/수정 패널의 월 선택 영역 근처
  • 동작: 다음 월을 지연 월로 추가하고, 해당 월도 실적 입력 대상에 포함합니다.
  • 예시: 10월 종료 KPI가 지연되면 11월 지연 월 추가, 필요 시 이후 12월도 추가
지연 월을 추가한 뒤에는 해당 월의 실적, 코멘트, 그래프 말풍선, 증빙을 동일하게 관리합니다.

10. 상황별 입력 예시

예시 1. 9월까지 총 5건 완료해야 하는 KPI
  • 평가 유형: 정량 평가
  • 측정 기준: 건
  • 측정 방향: 높을수록 좋음
  • 계산 기준: 누적 계산
  • 월별 목표값: 1월 1, 4월 2, 6월 1, 9월 1
  • 목표 말풍선: 1월 검토, 4월 제작, 6월 검증, 9월 완료 등 필요한 월만 입력

최종 목표값은 5건으로 자동 표시됩니다.

예시 2. 매월 불량률을 낮춰야 하는 KPI
  • 평가 유형: 정량 평가
  • 측정 기준: % 또는 PPM
  • 측정 방향: 낮을수록 좋음
  • 계산 기준: 당월 단독
  • 월별 목표값: 각 월 목표 불량률 또는 PPM 입력

낮을수록 좋은 KPI는 목표값이 그래프 중간에 오도록 표시되어 실적이 목표보다 낮은지 쉽게 볼 수 있습니다.

예시 3. 100% 달성을 목표로 하는 진행률 KPI
  • 평가 유형: 정성 평가 또는 정량 평가
  • 측정 기준: %
  • 측정 방향: 높을수록 좋음
  • 월별 목표값: 1월 10, 4월 40, 7월 70, 10월 100 등

높을수록 좋은 % KPI는 100%를 기준으로 그래프가 표시되며, 상단 라벨이 잘리지 않도록 화면 여백만 추가됩니다.

예시 4. 신규 설비 제작 KPI
  • B/M: 신규제작
  • 측정 기준: % 또는 건
  • 목표 말풍선: 3월 검토, 6월 제작, 8월 책임, 10월 양산
  • 실적 말풍선: 6월 고객 요청 지연, 8월 부품 입고 지연 등

B/M이 텍스트여도 그래프 막대는 최종 목표값 기준으로 표시됩니다.

11. FAQ / 주의사항

Q1. 목표 말풍선을 지웠는데 계속 보이면?

KPI 수정 화면에서 해당 월의 목표 말풍선 입력칸을 완전히 비운 뒤 저장합니다. 저장 후 그래프에서 목표 말풍선이 사라져야 합니다.

Q2. 실적 말풍선과 코멘트는 무엇이 다른가요?

그래프 말풍선은 막대 위에 짧게 보이는 한 줄 메모입니다. 코멘트는 상세 영역에 남는 긴 설명입니다.

Q3. 월별 목표값이 없는 월에도 실적을 넣을 수 있나요?

KPI 설정의 목표 공백 처리 방식에 따라 달라집니다. 목표 없는 달 제외를 선택하면 목표값이 있는 월 중심으로 관리하고, 직전 목표 유지를 선택하면 이전 목표를 이어서 적용할 수 있습니다.

Q4. 최종 목표값이 내가 입력한 값과 다르게 보이면?

누적 계산 KPI는 월별 목표값의 합계가 최종 목표값입니다. 당월 단독 KPI는 마지막 목표 월의 값이 최종 목표 기준으로 사용됩니다.

Q5. 저장 버튼 문구가 사람마다 다른 이유는?

역할별 승인 흐름이 다르기 때문입니다. 일반 사용자는 1차 승인 대기로 제출되고, 그룹장은 최종 승인 대기로 제출되며, 팀장은 즉시 승인 완료로 반영됩니다.

Q6. 완료 처리 후에도 실적을 수정할 수 있나요?

최종 완료는 KPI 항목이 완료되었음을 표시하는 상태입니다. 수정이 꼭 필요하면 관리자, 그룹장, 팀장 등 권한자에게 확인 후 처리하는 것을 권장합니다.

Q7. 첨부파일은 언제 올려야 하나요?

실적 저장 시점에 첨부하는 것이 원칙입니다. 특히 최종 완료를 누를 때는 마지막 실적과 첨부파일이 함께 저장되어야 합니다.

운영 팁: KPI를 새로 만들 때는 먼저 “당월 단독인지 누적 계산인지”를 정하고, 그 다음 월별 목표값을 입력하세요. 이 순서가 맞아야 그래프와 최종 목표값이 자연스럽게 맞습니다.

12. 빠른 체크리스트

KPI 추가 전

  • 이 KPI는 당월 단독인가, 누적인가?
  • 높을수록 좋은가, 낮을수록 좋은가?
  • 측정 기준 단위는 무엇인가?
  • 최종 목표월은 언제인가?
  • 월별 목표값을 어떤 월에 넣을 것인가?

실적 저장 전

  • 월 선택이 맞는가?
  • 실적값 단위가 목표값과 같은가?
  • 코멘트에 상세 설명을 남겼는가?
  • 그래프 말풍선은 짧게 입력했는가?
  • 증빙 파일을 첨부했는가?

최종 완료 전

  • 마지막 월 실적이 저장됐는가?
  • 첨부파일이 누락되지 않았는가?
  • 승인 상태를 확인했는가?
  • 지연 월 추가가 더 필요 없는가?
  • 완료 후 부서 화면에서 완료 표시를 확인했는가?
