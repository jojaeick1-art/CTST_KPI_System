# KPI 회귀 체크리스트

## 1) KPI 항목 등록
- 항목 추가 팝업에서 필수값 누락 시 필드 하단 에러가 표시된다.
- 기준이 `ppm`/`수량(k)`/`건`일 때 목표값이 없으면 저장되지 않는다.
- 저장 성공 시 목록에 신규 행이 보인다.

## 2) 가중치 검증
- 예: 현재 합계 95에서 5 입력 -> 저장 가능, 상태 `100/100 (정상)`.
- 예: 현재 합계 95에서 3 입력 -> 저장 가능, 상태 `98/100 (미배분)`.
- 예: 현재 합계 95에서 6 입력 -> 저장 불가, `100점 초과` 오류.
- 프론트 우회 요청 시에도 백엔드에서 100 초과를 거부한다.

## 3) 그룹 표시(대분류/소분류)
- 같은 대분류는 한 구간으로 묶여 보인다.
- 같은 대분류 내 같은 소분류도 하위 구간으로 묶여 보인다.

## 4) 자동완성
- 대분류 입력에서 기존 대분류 후보가 노출된다.
- 소분류 입력에서 기존 소분류 후보가 노출된다.

## 5) 월별 성과 그래프/범례 안내
- 목표는 빨강 실선, 실적은 파랑 막대로 보인다.
- 안내 문구가 차트 하단에 고정 노출된다.

## 6) 상/하반기 목표 연동
- 등록한 상반기/하반기 목표값이 상세 모달 차트 목표값 계산에 반영된다.
- 월별 실적 저장 시 해당 월 달성률이 갱신된다.

## 7) 접근성/사용성
- 툴팁 아이콘은 마우스 hover와 키보드 focus에서 안내를 제공한다.
- `저장 후 계속 입력` 활성화 시 모달이 닫히지 않고 다음 항목을 연속 등록할 수 있다.
